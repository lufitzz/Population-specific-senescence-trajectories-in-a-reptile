# Senescence diverges across thermal environments: Data and analysis code

Code repository accompanying the manuscript:

**Actuarial senescence and reproduction diverge across contrasting thermal environments in a wild ectotherm**  
LJ Fitzpatrick*, H Cayuela, M Olsson, GM While and E Wapstra  
*Ecology Letters* (pre submission)

---

## Overview

This repository contains R Markdown workflows for analysing actuarial and reproductive senescence in the spotted snow skink (*Carinascincus ocellatus*) across contrasting highland and lowland thermal environments in Tasmania, based on 25 years of longitudinal mark–recapture data.

---

## Repository structure
```
scripts/     R Markdown workflows (numbered in manuscript order)
data/        Subsetted data files required by each script
outputs/     Model outputs (BaSTA, BaFTA, etc.)
figures/     Final manuscript figures
```

## Scripts

Scripts 01–03 generate model outputs and can be run independently.  
Script 04 depends on outputs generated by scripts 01–03.

| Script | Description |
|--------|-------------|
| 01_Actuarial_senescence_BaSTA.Rmd | Survival and mortality modelling |
| 02_Reproductive_senescence_BaFTA.Rmd | Reproductive senescence modelling |
| 03_Growth_curve.Rmd | Von Bertalanffy growth curve analysis |
| 04_Manuscript_figures.Rmd | 2 Multipanel figures |

## Reproducibility

All analyses were conducted in R version 4.5.1.

To reproduce the results:

1. Run scripts 01–03 to generate model outputs  
2. Run script 04 to generate manuscript figures  

All scripts are fully annotated and can be run independently where indicated.
